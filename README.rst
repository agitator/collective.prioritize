==============================================================================
collective.prioritize
==============================================================================

Adds fields for duration and level of prioritization for content items.


Installation
------------

Install collective.prioritize by adding it to your buildout::

   [buildout]

    ...

    eggs =
        collective.prioritize


and then running "bin/buildout"
